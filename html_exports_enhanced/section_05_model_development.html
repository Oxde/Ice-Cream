5 Model Development

5.1 Overview

The model development process prioritized data-driven optimization over industry assumptions. Rather than applying blanket transformations across all media channels, we implemented an individualized approach that optimized adstock and saturation parameters for each channel based on their unique characteristics and relationship with sales.

5.2 Baseline Linear Model

We began with a multiple linear regression using raw media spend and control variables. This baseline model achieved modest performance but revealed key limitations:

  • Failed to capture media carryover effects
  • Ignored diminishing returns at high spend levels
  • Treated all channels identically despite different behaviors

These insights motivated our systematic approach to transformation optimization.

5.3 Data-Driven Adstock Optimization

Adstock transformation captures the reality that advertising effects persist beyond the initial exposure week. We optimized decay rates individually for each channel by maximizing correlation with sales.

Adstockt = xt + λ × Adstockt-1

The optimization revealed that traditional broadcast media showed stronger carryover effects while digital channels exhibited more immediate impact patterns.

5.4 Individual Saturation Curve Optimization

Saturation transformations model diminishing returns as spend increases. We tested six transformation types for each channel, selecting the one that maximized correlation with sales:

  • Linear: x/1000
  • Logarithmic: log(1 + x/1000)
  • Square Root: √(x/100)
  • Power 0.3: (x/1000)0.3
  • Power 0.5: (x/1000)0.5
  • Power 0.7: (x/1000)0.7

5.5 Final Model Specification

The final model combines all optimized transformations into a comprehensive framework:

Sales(t) = α + Σ βi · Si(Xadstock,i(t)) + Σ γj · Zj(t) + ε(t)

Where:

  • Si() = Channel-specific optimal saturation function
  • Xadstock,i(t) = Adstocked spend (where beneficial)
  • Zj(t) = Control variables (promotions, seasonality, weather)
  • βi, γj = Ridge regression coefficients (α = 1.0)

5.6 Model Performance

The data-driven optimization approach yielded significant improvements over the baseline model:

Final Model Metrics

  • Test R²: 59.3% (good predictive accuracy)
  • Test MAPE: 14.4% (reasonable error margin)
  • Overfitting Gap: 0.077 (minimal, indicating good generalization)

Technical Note: All transformations were applied consistently across training and test sets. Feature standardization was performed after transformations to ensure proper Ridge regression behavior.

5.7 Key Methodological Advantages

Our approach offers several advantages over traditional MMM methodologies:

  1. No Arbitrary Assumptions: Parameters derived from data, not industry rules of thumb
  2. Channel Individuality: Each channel optimized based on its unique characteristics
  3. Business Realism: Transformations align with observed channel behaviors
  4. Validation Rigor: Proper train/test split ensures reliable performance estimates

This data-driven methodology provides a robust foundation for understanding true channel contributions and optimizing media allocation decisions.
